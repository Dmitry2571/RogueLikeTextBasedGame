# RogueLikeTextBasedGame
Python text-based game.
The code of this game demonstrates the principles of object-oriented programming.

Current version: 0.0.3c

In the current version, the following are available: 
Locations:
- SlumsArena
- SlumsOfUlayim

Acievements, quest, arena combat.

The game is played through direct communication, instead of going by numbers, as in many other games.
